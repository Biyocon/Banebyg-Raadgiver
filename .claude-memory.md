# .claude-memory.md — BBTR Project Context
 
## Project Identity
 
- **Product:** BaneByg Teknisk Rådgivning (BBTR)
- **Organization:** Banedanmark, Anlægssektionen
- **Platform:** SharePoint Online (SPFx + React)
- **Repo:** Banebyg-R-dgiver
 
## Locked Product Decisions
 
- BBTR is a **superset of BBE** (BaneByg Entreprise) — shared SPFx codebase, toggled via `IS_BBTR` feature flag
- **4 sections:** Poster, Udbud, ATR & Bemanding, Bilag
- **13 SharePoint lists:** 7 shared with BBE + 6 BBTR-unique
- **25 fagposter** (vs BBE's 17), including phase-based posts 22–25
- **Fase dropdown** replaces BBE's manual stages; no Delentreprise
- **4 roles:** Administrator, Projektleder, Bruger, Læser
- Poster and Bilag sections **admin-only** access
- All UI, field names, code comments in **Danish**
 
## Locked Technical Decisions
 
- SPFx (SharePoint Framework) with React for all custom webparts
- PnPjs for SharePoint list CRUD operations
- Azure AD for authentication
- SharePoint Lists as sole data store (no external DB)
- Power Automate for server-side automation (ATR generation, versioning)
- Document generation deferred to Phase 9 (Azure Function + Office templates)
